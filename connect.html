How to Set up a Push command from Cloud9 to Github Step 1: Create a Repo in github Step 2: Copy URL from github Step 3: On Cloud9 (Add-On Services) Click activate on githup Step 4: Use URL from github to link Step 5: Once your Cloud9 dashboard appears, look under Projects on Github, click repo Step 6: Select Clone project Step 7: Your Repo will now appear under My Projects. Click the arrow to start editing workspace. Step 8: Create a new file with a .html extenstion and save Step 9: To upload this file to github enter the following information to the terminal git status git add . "alert.html git commit -m "Message Here" git push origin master Step 10: Success!!
